Einführung

Willkommen im Dashboard von 'The Moist Soil Society'. Diese Applikation dient der Echtzeit-Überwachung der Bodenfeuchtigkeit deiner Zimmerpflanzen. Die Daten werden im Intervall direkt vom Server abgefragt. Sollte der gemessene Wert den von dir definierten Grenzwert unterschreiten, wechselt die jeweilige Pflanzenkarte in den Alarmzustand (Rot). Neue Sensoren können unten im Formular dynamisch hinzugefügt werden.
